簡介

這是郭耀仁在台灣大學資訊工程學系資訊系統訓練班所開設的 R 語言新手村課程網頁，課程時數 10 小時。

課程大綱

  • 建立你的 Python 開發環境
  • 基礎 Python 資料型態
  • 基礎 Python 語法
  • 套件與函數
  • Numpy 套件
  • Matplotlib 套件
  • 使用 Pandas 套件進行資料探索（Exploratory Data Analysis）
  • 使用 Pandas 套件進行資料整理
  • 建立一個預測模型
  • 實戰演練
